# Roshan Chimney Service Website

A responsive website for **Roshan Chimney Service** that helps customers easily book chimney cleaning, repair, and installation services online.

This project provides a simple and user-friendly interface where users can explore chimney services, view contact information, and request maintenance or repair services.

---

## Features

- Responsive design for mobile and desktop
- Service information for chimney cleaning and repair
- Easy contact and booking section
- Clean and simple user interface
- Fast loading website

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
